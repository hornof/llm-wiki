## Key Claims / Takeaways

### The 4 pathways (Legg framework)

The framework identifies **4 distinct technical pathways** from AGI to superintelligence, each with named technical bottlenecks:

| # | Pathway | Bottleneck class |
|---|---|---|
| 1 | **Scaling** | Sample efficiency + compute + data exhaustion |
| 2 | **Recursive self-improvement (RSI)** | First-few-self-improvement-steps; loop-closure |
| 3 | **Multi-agent collectives** | Coordination + emergent-coherence |
| 4 | **(4th pathway)** | Verification-pending specifics |

**First wiki-captured 4-pathway canonical taxonomy for AGI → superintelligence transition**. Note: pathways are **not equivalent paths** — *"which bottleneck breaks first determines which world we get, not whether we get superintelligence"* (Luke-side insightful framing).

### Luke-side hot-take framing

> *"The gap from AGI to superintelligence isn't compute or algorithms — it's recursion. If self-improvement actually works, the hard part is the first few steps. After that, it's no longer a human problem."*

### Luke-side insightful framing

> *"Legg's four pathways frame superintelligence as a transition problem, not an arrival problem. That resets the question: which bottleneck breaks first determines which world we get, not whether we get superintelligence. Multi-agent collectives vs. recursive self-improvement aren't equivalent paths."*
