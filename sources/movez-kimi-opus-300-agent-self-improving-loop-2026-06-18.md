### The 10-step canonical playbook

| Step | Action | Canonical framing |
|---|---|---|
| **1** | Write a **spec, not a prompt** | *"A one-line prompt is a wish. A spec is an order. The swarm executes orders."* + canonical spec template (GOAL + SCOPE + RULES + SOURCES + OUTPUT + ON CONFLICT + STOP CONDITION) |
| **2** | Read the **decomposition plan** before you spend a cent | Kimi shows execution plan: agent count + dependency order + step budget; *"checking the plan costs nothing"* |
| **3** | **Let it be wasteful** — that's the point | 300 sub-agents fire in parallel waves; each works in own bounded context window (avoids single-agent context-window collapse); cheap-volume-changes-what-you're-willing-to-attempt canonical-framing |
| **4** | Demand **real files, not a chat answer** | Output is structured deliverables (PDFs + spreadsheets + datasets + decks + working code); *"A comprehensive report' gives agents permission to stop early. '40-page PDF + one CSV with 20,000 rows + 14 export-ready PNG charts' gives them a quality target to hit"* |
| **5** | Point the **honest model at the output** and ask what's wrong | Opus 4.8 verify-gate; *"Its only job here is to refute, not to praise"* |
| **6** | **Save the whole workflow as a Skill** | *"The first run takes 20 minutes. Every run after it takes 30 seconds."* — canonical-cost-collapse anchor; saves input format + agent steps + output format |
| **7** | **Feed your own documents in as swarm knowledge** | Document-to-Skill captures domain (vs Skill captures process); *"every PDF, transcript, or spreadsheet you feed in becomes context that all 300 parallel agents can ground against"* |
| **8** | **Turn the verify feedback into a permanent rule** — `CONSTRAINTS.md` canonical-pattern | *"The drift that Opus flagged on run #1 becomes a hard rule on run #2"*; canonical-living-documentation framing |
| **9** | **Replay the skill on new inputs** — watch the cost collapse | *"20 minutes on run one. 30 seconds on run fifty."* canonical-cost-collapse-on-invoice anchor |
| **10** | **Promote the loop to a background agent** | Schedule-trigger / file-drop-trigger / monitored-URL-trigger; *"only ping me if a deviation crosses [threshold]"* |
