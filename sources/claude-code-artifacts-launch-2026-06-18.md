## Key Claims / Takeaways

### The launch

- **Source**: [Anthropic announcement](https://claude.com/blog/artifacts-in-claude-code) (2026-06-18)
- **Feature**: **Claude Code artifacts** — captures work progress as live, shareable visual pages
- **Built on**: full session context (codebase + connectors + conversation)
- **Updates**: live page refreshes in place; teammates see updates the moment they're published
- **Versioning**: every publish is a new version at the same link; full version history; gallery management
- **Availability**: beta to **Claude Team + Enterprise orgs**; Claude Code CLI + desktop app
- **Privacy**: **private to organization by default** — cannot be made public; admin org-level toggle + role-based scoping + retention policies + compliance API
