The people getting insane results from Claude Projects are building something completely different. They are building customized AI employees. ### Why Most Claude Projects Fail (3-reason framework)

1. **System prompt too vague** — *"'You are a helpful writing assistant' is not a system prompt. It is a wasted opportunity."*
2. **No knowledge files uploaded** — *"A project without knowledge files is just a labeled conversation."*
3. **Project tries to do everything** — *"One project called 'Work Stuff' that handles emails, content, strategy, research, coding, and analysis will produce mediocre results across all of them. Separate your workflows into separate projects. Each one should do one thing extremely well."*

**First wiki-captured 3-failure-mode framework for Claude Projects** — parallels [[trq-dynamic-workflows-harness-2026-06-02|Thariq's 3-failure-mode framework for coding agents]] + [[anthropic-self-service-data-analytics-2026-06-03|Anthropic data-analytics 3-failure-mode framework]]. **Third wiki-captured 3-failure-mode framework at the Claude-primitive layer** within 5 days.

### The 6-Part Blueprint

| Part | Purpose | Example |
|---|---|---|
| **1. Identity Block** | Tells Claude who it is in this project | *"You are my senior content strategist... working with me for two years"* |
| **2. Rules Block** | Non-negotiable ALWAYS / NEVER list (*"the most important part by far"*) | *"ALWAYS: short paragraphs, specific numbers... NEVER: 'leverage', 'utilize', 'streamline'"* |
| **3. Process Block** | How Claude should approach work (not what to produce) | *"Before writing... think about what the reader already believes... Outline... Write first draft in one pass... Review against Rules... Read as a busy person scrolling"* |
| **4. Output Format Block** | Exact finished-product specification | *"Title format... Length 2000-3500... Structure: bold contrast opener... Closing: italic line + bold italic"* |
| **5. Knowledge Files** | 5 files: Style guide (REQUIRED) + Audience profile (REQUIRED) + Competitor analysis (RECOMMENDED) + Performance data (RECOMMENDED) + Template library (RECOMMENDED) | Claude pattern-matches voice from style guide; calibrates content from audience profile |
| **6. Onboarding Message** | First message in new conversation — "daily briefing you would give an employee" | *"Today we are working on \[TASK\]... Reference my style guide... Tell me: 1. angle 2. challenged belief 3. one-sentence hook"* |
