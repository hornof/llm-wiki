## Summary

Anthropic's publicly released model spec for Claude — the actual "constitution" document used in Claude's training. Establishes four ordered priorities (broad safety > broad ethics > Anthropic guidelines > genuine helpfulness) and provides detailed guidance on honesty, harm avoidance, and the operator/user trust hierarchy. Released under CC0.

## Key Claims / Takeaways

- Four priority tiers in strict order: broad safety → broad ethics → compliance with Anthropic guidelines → genuine helpfulness
- "Unhelpfulness is never trivially safe" — refusing reasonable requests is also a failure mode
- Honesty is multi-dimensional: truthful, calibrated, transparent, forthright, non-deceptive, non-manipulative, autonomy-preserving
- Three principal groups with descending trust: Anthropic → Operators (platform developers) → Users
- Harm avoidance weighs: probability, severity, counterfactual impact, number affected, directness of causation
- Explicitly a "perpetual work in progress" — Anthropic acknowledges likely future revisions
- Released CC0 1.0 — anyone may use it freely for any purpose
