## Summary

**Prasanna S** (@myprasanna, ex-cofounder/CTO of Rippling) launches **[[vorflux|Vorflux]]** (@vorfluxai), a cloud **"autopilot for software engineering,"** with a **$15M YC-led seed**. The thread frames a hard autopilot-vs-copilot distinction and a "Great Flattening" company-shape thesis, and locates the durable value in a **model-neutral harness**. Launch-day self-promotional thread (with a $200-credits offer) — founder pedigree + raise are the hard signals; capability claims are self-asserted.

## Key Claims / Takeaways

- **Autopilot vs copilot** (thesis 1/): *"Every AI coding tool still makes you fly the plane… you stay in the seat, approving every turn. The models quietly got good enough to fly the whole route, but the tools never caught up. So we built the autopilot."*
- **The models "woke up"** (2/): *"The models got genuinely superhuman at code this year. GPT-5.6 Sol just topped the coding benchmarks; a different lab led the month before; no single model sweeps."* And they *"outgrew your laptop"* — hence cloud execution.
- **The Great Flattening** (3/): *"Code stopped being the hard part this year… planning, design, review, execution all collapse toward the system that runs the work. One person inside, or zero."*
- **The harness** (4/): *"That judgment needs somewhere to live. It lives in the harness: the one place your engineering principles run on every session, lossless and compounding. We train no models, so it stays neutral. The labs ship their opinion. Nobody ships yours."*
- **Funding**: $15M seed — **Y Combinator**, Peak XV Partners, Alliance DAO + angels (Parker Conrad, Balaji Srinivasan, Naval Ravikant, Matt Shumer, and others).

## Why it matters

- **A credible-founder entrant** in the autonomous-coding-agent race, explicitly positioned against the [[claude-code|Claude Code]]/[[codex|Codex]] *copilot* register with an *autopilot* claim.
- **Vendor-side [[loop-engineering|harness-primacy]] datapoint**: "judgment lives in the harness… lossless and compounding… we train no models, so it stays neutral" is a clean statement of the harness as the durable, model-agnostic layer — and rhymes with [[reverse-information-paradox|Nadella's "Choice"]] (decouple orchestration from any single model).
