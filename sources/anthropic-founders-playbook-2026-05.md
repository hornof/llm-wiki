## Key Claims / Takeaways

### What the playbook contains

- **Validate a problem hypothesis, map a competitive landscape, run customer discovery with AI**
- **Architecture, scope, and security practices** that keep AI-generated MVP codebases from accruing technical debt
- **A measurement framework for distinguishing genuine product-market fit from early hype** — pairs structurally with [[willison-anthropic-openai-pmf-2026-05-27|Willison's API-PMF-vs-end-user-product-PMF distinction]]
- **A Launch-stage operating system that replaces founder attention with agentic workflows**
- **A product matrix for when and how to use [[claude-desktop|Chat]], [[claude-cowork|Claude Cowork]], and [[claude-code|Claude Code]]** across each stage — first wiki-captured *Anthropic-canonical* product-matrix recommendation for which Claude surface fits which startup stage
