## What It Is
A Claude Code plugin companion to [[claude-obsidian]] that adds an AI-orchestrated visual canvas layer. 12 templates, 6 layout algorithms, AI image generation, and full canvas orchestration. Auto-detects claude-obsidian vaults.

## Traction Signals
- Companion to [[claude-obsidian]] (2,626 stars); bundled recommendation in that repo — [[github-claude-obsidian]]

## How to Use It
```bash
claude plugin install AgriciDaniel/claude-canvas
```
Use `/canvas` commands to add images, text cards, PDFs, and wiki pages to Obsidian canvas files with auto-positioning inside zones.

## Key Concepts
- **Templates**: 12 pre-built canvas layouts (knowledge graphs, presentations, flowcharts, mood boards)
- **Layout algorithms**: 6 algorithms for auto-positioning nodes
- **Zones**: labeled areas that organize content spatially on the canvas
- **banana integration**: captures recently AI-generated images onto the canvas

## Compared To
- [[claude-obsidian]]'s built-in `/canvas`: claude-canvas is the advanced version with more templates and layout intelligence
- Excalidraw (Obsidian plugin): manual drawing; claude-canvas is AI-orchestrated
