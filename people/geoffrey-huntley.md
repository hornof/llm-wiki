## Who They Are

Geoffrey Huntley is the independent practitioner who **published the ralph loop pattern** in July 2025 ([ghuntley.com/ralph/](https://ghuntley.com/ralph/)) — the 3rd stage in the **5-stage Loop Engineering lineage** captured by [[mvanhorn-wtf-is-a-loop-2026-06-07|Van Horn's WTF Is a Loop synthesis]]. **First wiki entity surface 2026-06-08** via Van Horn's framing of ralph as the load-bearing pre-/goal practitioner-content milestone.

## Their Current Focus

The **ralph loop** as a discipline of context-reset-per-iteration:
- Bash one-liner piping the same prompt file into the coding agent repeatedly
- Each iteration **resets the context to a fixed set of anchor files** instead of letting the conversation grow
- Assumes terminal stays open (pre-orchestration-loop generation; per Van Horn: *"the 2026 version assumes it does not"*)

Per [[mvanhorn-wtf-is-a-loop-2026-06-07|Van Horn]]: *"It is almost insultingly simple, a bash one-liner that pipes the same prompt file into the agent over and over. Its real innovation was discipline: every iteration resets the context to a fixed set of anchor files instead of letting the conversation grow."*
