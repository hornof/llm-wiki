## What It Is
Internet infrastructure and security company. Operates a global edge network spanning CDN, DNS, DDoS protection, Workers serverless compute, R2 storage, and a growing AI-agent infrastructure surface. First wiki appearance in May 2026 via the Stripe Projects co-launch — concrete agent capability crossing into account creation and financial transactions.

## Key Products (relevant to this wiki)
- **Cloudflare Workers** — edge serverless runtime; broadly used as a deployment target for agent-built apps
- **Cloudflare Registrar** — domain purchase surface; now reachable by agents via Stripe Projects
- **Code Mode MCP server** — Cloudflare-hosted [[mcp]] server enabling agent-driven code execution and infra provisioning — [[cloudflare-stripe-projects-agents-2026-05]]
- **Cloudflare Agent Skills** — Cloudflare's named surface for skill-style agent extensions; first wiki mention via the Stripe Projects launch

## Traction Signals
- 2026-05-06: **Stripe Projects co-launch** — protocol co-designed with Stripe so agents can autonomously create Cloudflare accounts, purchase domains via Cloudflare Registrar, and deploy apps with issued API tokens. Stripe attests user identity and orchestrates payment with default $100/month per-provider spending cap. Cloudflare offers $100,000 in Cloudflare credits to startups incorporating through Stripe Atlas. Authors: Sid Chatterjee, Brendan Irvine-Broque. — [[cloudflare-stripe-projects-agents-2026-05]]
- Cloudflare's positioning as agent-infrastructure-first (Code Mode MCP, Agent Skills, Stripe Projects) is an explicit answer to the [[saas-disruption-thesis]] question of how agents purchase and consume SaaS — they need protocol-level identity + payment primitives, and Cloudflare is co-defining them with Stripe.
