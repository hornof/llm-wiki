## Academic Publications

### Self-Specializing Mobile Code (2000)
- **Venue**: Active Networks: International Working Conference (IWAN), Springer
- **Authors**: L. Hornof
- **Google Scholar**: https://scholar.google.com/scholar?cluster=17260482565727073218
- **Abstract**: Introduces self-specializing mobile code — programs transmitted over a network that automatically customize themselves to their destination. Combines program specialization and program verification to achieve higher performance without compromising safety. Demonstrated via FAST-net (Fast, Adaptive, Safe, Typed active network), showing adaptive network services can run over 4x faster than normal network services.
- **Context**: PhD-era work; compilers / active networks domain; predates current AI focus.
