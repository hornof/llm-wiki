### AI / Deep Learning

**Flexpoint: An adaptive numerical format for efficient training of deep neural networks** (2017)
- **Authors**: U Köster, T Webb, X Wang, M Nassar, AK Bansal, W Constable, O Elibol, L Hornof, et al.
- **Venue**: Advances in Neural Information Processing Systems 30 (NeurIPS)
- **Citations**: 388
- **Summary**: Proposes Flexpoint, an adaptive fixed-point numerical format for DNN training that dynamically manages the exponent across a tensor. Targets efficient hardware execution. Nervana/Intel era work.

**Dynamic management of numerical representation in a distributed matrix processor architecture** (2020)
- **Authors**: U Koster, WH Constable, LJ Hornof, CK Kloss, A Khosrowshahi, et al.
- **Venue**: US Patent 10,552,119
- **Citations**: 70
- **Summary**: Patent covering dynamic numerical representation management in distributed matrix processor (NNP/Nervana hardware). Intel-era work.

### Active Networks / Mobile Code

**Self-specializing mobile code for adaptive network services** (2000)
- **Authors**: L Hornof
- **Venue**: IFIP International Working Conference on Active Networks (IWAN), 102–113
- **Citations**: 6
- **Summary**: Programs transmitted over a network that automatically customize themselves to their destination. Demonstrated via FAST-net (Fast, Adaptive, Safe, Typed). Adaptive services run 4x faster than static ones without compromising safety.
